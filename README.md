# Hackaton partie web

## Techno

Le projet est sous symfony donc php/twig

## Requirements

- Symfony : https://symfony.com/doc/current/setup.html
- Composer : apt-get install composer (1.10.24)
- php 7.2.5 +

## Demarrage

- git pull
- composer install
- symfony server:start
